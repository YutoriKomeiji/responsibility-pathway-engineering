初めて読む人のための装飾付きマップです。何を読み、何を試し、強い主張の前に責任がどこへ戻るべきかを確認します。

English 日本語カタログ
境界: このページは静的な閲覧・レビュー補助です。認証、法的レビュー、安全性レビュー、遵法性レビュー、公平性レビュー、本番承認、またはAIへの最終責任移転ではありません。

責任経路の中核

人間の依頼

人間または組織が作業を開始し、判断主体への戻り先を保ちます。

AIが支援する

AIは草案作成、要約、提案、証拠準備を支援できますが、最終責任主体にはなりません。

証拠を見える状態にする

source reference、仮定、除外事項、不確実性を確認可能に保ちます。

人間の承認へ戻れる

承認、停止権限、return pointが人間または組織の判断へ接続されている必要があります。

修復と再開を可能にする

経路が不完全、ノイズ混じり、または誤りを含む場合、repair ownerへ戻せる状態にします。

次に作れるページ

  1. Example browser: 記入済みYAML examplesを視覚的に確認するページ。
  2. Template helper: responsibility-path recordを組み立てる非送信型フォーム。
  3. Boundary glossary: 除外主張と非認証表現をまとめる装飾ページ。
  4. Reviewer checklist: quickstartをブラウザで使いやすくしたチェックリスト。

主張境界: 装飾ページは導線を見やすくするためのものです。成熟度、証明力、安全性、合法性、遵法性、公平性、本番Ready、責任移転を強めるものではありません。

すべての定義と主張について、人間の著作者が責任を持ちます。AI toolsは草案作成や実装を支援しますが、責任を引き受けません。
