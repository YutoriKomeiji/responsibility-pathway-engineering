How to read this page

Source files remain authoritative

This page groups examples for easier browsing. It does not replace the source YAML, JSON, repository docs, or human review.

Checker passes are bounded

Use checker coverage before interpreting any pass. A pass is a bounded repository-maintenance signal only.

Example index gives detail

Use example index for detailed purpose, boundary, and reading notes.

Human author remains responsible for definitions and claims. AI tools may assist navigation, drafting, and implementation but do not become responsible authors.
