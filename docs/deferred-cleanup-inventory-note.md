# Deferred Cleanup Inventory Note

This note records a lightweight handling rule for deferred cleanup items.

## Rule

Do not let deferred cleanup items disappear.

A fixed review cycle has not been selected yet.

Until a cycle is selected, keep deferred cleanup items visible as inventory.

## Examples

Deferred cleanup items may include:

- links that could not be added to a heavier navigation file;
- references that were moved to a lighter note;
- files treated as temporarily hard to update;
- follow-up consolidation tasks.

## Boundary

This is an inventory note only.

It does not schedule reviews, create automation, change repository behavior, or change any schema, checker, workflow, runtime, connector, or formalization file.
