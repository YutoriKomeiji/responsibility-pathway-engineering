# Zenn Level 2 Publication with Parallel Cleanup

This note records the current publication-maintenance policy for Zenn Level 2 repository walkthrough readiness.

It is a planning note only. It is not an article draft, title decision, publication record, file deletion decision, certification record, conformance claim, production approval, standardization claim, legal review, safety review, compliance review, fairness review, social acceptance proof, or AI final-responsibility transfer mechanism.

## Policy

Level 2 repository-walkthrough preparation may continue while file-organization audit runs in parallel.

File organization concerns should not block Level 2 unless they create a direct reader-path contradiction, broken public entrance, missing non-claim boundary, or misleading publication-facing statement.

## Cleanup posture

Use `docs/repository-file-organization-audit.md` for file-sprawl, overlap, merge, deprecation, or delete-candidate inspection.

Do not begin broad cleanup solely because Level 2 publication is approaching.

Prefer:

- classify first
- preserve restart paths
- keep alternate routes visible
- defer deletion
- avoid rewriting long files only for neatness

## Level 2 remains focused

Level 2 should only require that readers can find:

- public entrance
- operation path
- current task inventory
- progress map
- roadmap position
- example index
- checker coverage
- external-review readiness path
- social-connection review path
- gap inventory
- non-claims and deferred boundaries

Level 2 does not require completing repository-wide cleanup.

## Stop condition

Stop Level 2 publication preparation only if a file-organization issue makes the repository walkthrough misleading.

Examples:

- public entrance points to stale or contradictory state
- reader cannot reach checker boundaries
- non-claims are hidden or contradicted
- an active file sounds like certification, conformance, legal validity, safety proof, social acceptance, production readiness, or AI final-responsibility transfer

## Next safe action

Keep this note connected to Zenn Level 2 readiness and file organization audit while continuing repository-side Level 2 preparation.
