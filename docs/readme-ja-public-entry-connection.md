## なぜこの文書があるか

そのため、日本語公開記事、特に Zenn から来る読者向けの短い導線は `docs/zenn-publication-ja-reader-note.md` に切り出しました。

この文書は、その切り出しが意図的な reader-path operation であることを記録します。
