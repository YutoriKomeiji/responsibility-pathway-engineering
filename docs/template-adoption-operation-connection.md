## Operational rule

The template route should remain easy to find, but it must not imply finished implementation maturity.

A reader should be able to understand:

- what the template is for
- how to copy or adapt it
- that it is under construction
- that it is non-certifying
- that AI does not become the final responsibility holder
- where human or institutional responsibility can return
