Keep the public template route visible from the README files and connected to Level 2 readiness.

Keep template language short, practical, and non-certifying.

Keep `templates/README.md` as the detailed template index so root README files do not become too large.

## P2 bounded artifact preparation

Additional template artifacts may be added only if they remain:

- small
- copyable
- under construction unless explicitly stabilized later
- human-readable
- non-certifying
- responsibility-return preserving
- clear about human or institutional review
- clear about excluded claims

- GitHub issue responsibility-path template
